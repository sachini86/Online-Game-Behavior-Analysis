# Online Game Behavior Analysis

Analyze online player behavior to uncover engagement patterns, spending trends, and actionable insights using Python and data visualization.

---

## **Project Overview**
This project explores and analyzes online gaming behavior using a dataset of player activities. The goal is to extract insights on player engagement, preferences, and in-game spending to help game developers enhance user experience and retention.

---

## **Dataset**
The dataset includes the following columns:

- **PlayerID**: Unique identifier for each player
- **Age**
- **Gender**
- **Location**
- **GameGenre**
- **PlayTimeHours**
- **InGamePurchases**
- **GameDifficulty**
- **SessionsPerWeek**
- **AvgSessionDurationMinutes**
- **PlayerLevel**
- **AchievementsUnlocked**
- **EngagementLevel**

---

## **Objectives**
- Identify player demographics and behavior trends  
- Analyze playtime, in-game purchases, and achievement patterns  
- Segment players based on engagement and spending  
- Provide actionable recommendations for game improvement  

---

## **Tools & Technologies**
- Python: Pandas, NumPy, Matplotlib, Seaborn  
- Jupyter Notebook  
- Git & GitHub  

---

## **Analysis Workflow**
1. Data Cleaning & Preprocessing  
2. Descriptive Statistics & Visualization
